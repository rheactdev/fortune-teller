# 🔮 Fortune Teller (Audentus)

This repository contains the **Audentus** fortune-telling application.

## 📂 Project Structure

- **`vite-refactor/`**: The modern, Vite-powered version of the application. This is the primary development directory.

## 🚀 Quick Start

To get started with the main application, head over to the [vite-refactor](file:///Users/donut/Desktop/Programs/fortune-teller/vite-refactor/README.md) directory:

```bash
cd vite-refactor
pnpm install
pnpm dev
```

For more details on features and configuration, see the [detailed README](file:///Users/donut/Desktop/Programs/fortune-teller/vite-refactor/README.md) inside the `vite-refactor` folder.
